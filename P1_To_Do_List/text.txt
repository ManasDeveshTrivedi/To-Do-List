The To-Do List Web App is a simple and interactive application built using HTML, CSS, and JavaScript. It helps users manage their daily tasks by allowing them to add, delete, and mark tasks as completed. Users can also set deadlines for tasks and view the remaining time in hours. The app automatically saves data using local storage, so tasks are not lost after refreshing the page. It also sorts tasks to keep unfinished items at the top for better organization.
The project uses basic frontend technologies, where HTML provides the structure, CSS is used for styling with a modern glassmorphism design, and JavaScript handles functionality and DOM manipulation. It also demonstrates key concepts such as event handling, local storage, and responsive layout design.
This project shows a strong understanding of frontend development and problem-solving. It can be further improved by adding features like dark mode, mobile responsiveness, notifications, analytics, and user authentication, or by extending it into a full-stack or machine learning-based application.
